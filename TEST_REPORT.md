## 1. 测试概述

### 1.1 项目简介

知伴教师端是一款面向教师的在线教育管理平台，主要功能包括班级管理、视频上传与审核、学情数据分析等。

### 1.2 技术栈

| 类型 | 技术 |
|------|------|
| 前端框架 | React 19.2.5 |
| 构建工具 | Vite 8.0.9 |
| UI 组件库 | Ant Design 6.3.6 |
| 路由 | React Router DOM 7.14.2 |
| HTTP 客户端 | Axios 1.15.2 |
| 图表 | ECharts 6.0.0 |

### 1.3 功能模块清单

| 序号 | 模块名称 | 功能点数量 |
|------|----------|------------|
| 1 | 登录/注册 | 8 |
| 2 | 首页 | 2 |
| 3 | 班级管理 | 4 |
| 4 | 视频管理 | 5 |
| 5 | 待审核 | 2 |
| 6 | 数据分析 | 6 |
| 7 | 个人中心 | 3 |
| **合计** | - | **30** |
